## Features
 
- **Animated score rings** вҖ” Sleep, readiness, and activity scores with smooth fill animations

- **AI-powered recommendations** вҖ” Personalized nutrition, workout, and recovery plans generated by Claude API
- **DOI-cited research** вҖ” Every suggestion links to a real peer-reviewed study
- **Daily focus insights** вҖ” Smart summaries based on your current biometric state
- **Streak tracking** вҖ” Stay motivated with consecutive day tracking

## How It Works
 
**1. Upload** вҖ” User exports their Oura Ring data (JSON/CSV) and uploads through the mobile app.
 
**2. Parse** вҖ” S3 upload triggers the Parser Lambda, which extracts sleep stages, HRV, resting heart rate, body temperature, and activity metrics into DynamoDB.
 
**3. Recommend** вҖ” The Recommender Lambda reads the user's last 7 days of metrics, calculates trends (improving/stable/declining), and sends everything to the Claude API. Claude returns personalized nutrition, workout, and recovery plans вҖ” each with a DOI citation to a peer-reviewed study.
 
**4. Track** вҖ” Users mark each recommendation as followed or skipped. Over time, this enables trend analysis: *"Did the magnesium actually improve deep sleep?"*
 
## The App
 
The mobile app is built with React Native (Expo) and features a premium dark health-tech UI.
 
**Upload Screen** вҖ” Guided file upload with step-by-step parsing progress animation.
 
**Dashboard** вҖ” Animated score rings for sleep, readiness, and activity. Metric cards with 7-day sparkline trends. Daily focus insight card. Streak tracking.
 
**For You** вҖ” AI-generated recommendation cards organized by nutrition, movement, and recovery. Each card expands to show the full recommendation, the biometric trigger, the DOI citation, and action buttons.
 
## Lambda Functions
 
### Parser (`lambda/parser/handler.py`)
Triggered by S3 `ObjectCreated` events. Supports both JSON and CSV Oura export formats. Extracts:
- **Sleep** вҖ” deep/REM/light duration, efficiency, latency, HRV, heart rate
- **Readiness** вҖ” score, temperature deviation, resting HR, recovery index
- **Activity** вҖ” steps, calories, sedentary time, MET average
### Recommender (`lambda/recommender/handler.py`)
Called via API Gateway `POST /recommendations`. Queries DynamoDB for 7-day metrics, calculates trend direction, and generates personalized plans via Claude API. Every recommendation includes a specific actionable suggestion, the biometric trigger, and a DOI reference.
 
### Tracker (`lambda/tracker/handler.py`)
Called via API Gateway `POST /track`. Logs user actions against specific recommendations for longitudinal analysis.
 
## Quick Start
 
### Prerequisites
- AWS CLI configured with credentials
- Terraform >= 1.5
- Node.js >= 18 (for the mobile app)
- An Anthropic API key
### Deploy the Backend
 
```bash
git clone https://github.com/jessicaparadise/attune.git
cd attune
 
aws ssm put-parameter --name "/attune/claude-api-key" --type SecureString --value "key" --region us-west-2
 
cd terraform
terraform init
terraform apply
```
 
### Run the Mobile App
 
```bash
cd mobile
npm install
npx expo start -w
```
 
### Test the Pipeline
 
```bash
aws s3 cp sample-oura-export.json s3://attune-dev-oura-uploads/uploads/test-user-1/export.json
 
curl -X POST $(terraform output -raw api_url)/recommendations \
  -H "Content-Type: application/json" \
  -d '{"user_id": "test-user-1"}'
```
